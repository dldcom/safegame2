# 1막 — 화재 대피 설계서

> 1막 시나리오·체크포인트·자산·동선의 진실의 원천.
> Phaser 씬 구현 시 이 문서를 따라가면 됨.

---

## 한 줄 시놉시스

방과 후 도서관에서 친구 5명과 책을 읽던 중, 옆 교실에서 화재 발생.
**친구들은 우왕좌왕**, 학생 본인이 침착하게 매뉴얼대로 행동해 모두 운동장으로 안전하게 대피.

**핵심 메시지**: 위급 상황에서는 *안다고 생각하는 것* 보다 *실제로 침착하게 행동하는 것* 이 중요. 친구가 알려주는 게 아니라 **내가 침착해야 한다**.

## 캐릭터 — 6마리 동물 (범용 라이브러리)

### 핵심 변경
- **모두 캐릭터 (player 후보)** — `student`/`friend_*` 구분 X
- 시작 화면에서 학생이 1마리 고름 → player. 나머지 5마리 자동으로 NPC 슬롯 (`npc_friend_1` ~ `npc_friend_5`) 에 매핑
- 모든 캐릭터 **평범한 정자세 + 친근한 표정** (무서움 X)
- 무서워하는 표정/자세는 게임 코드에서 임시 효과 (몸 흔들림 tween 등)
- 다른 게임에도 그대로 재사용 가능한 **범용 캐릭터 라이브러리**

### 6마리 카탈로그 (`shared/lib/characters.ts`)

## 6 체크포인트 흐름 (확정)

### Map 1: 도서관 (`act1_library`)

#### CP1 — "불이야!" + 119 신고
| | |
|---|---|
| 트리거 | 게임 시작 후 3초. 옆 교실 창문에서 빨간 불빛 + 검은 연기 |
| 친구 반응 (무서워하기) | 친구1: "어어 저건 뭐야?" / 친구2: 책 떨어뜨림 / 친구3: "어떡해 어떡해" |
| 정답 행동 | A 버튼 → 휴대폰/벨 인터랙션 → 119 메뉴 → "불이야!" 외치기 (정확한 순서: 119 → 외치기) |
| 카운트다운 | 10초 |
| 오답/시간초과 시 힌트 | 친구 (랜덤 슬롯): "어어 어떡해... 신고? 뭔가 신고해야 한다고 배운 것 같은데..." |
| 정답 후 친구 반응 | 친구1: "맞아! 신고!" / 친구2: "너 진짜 침착하다" |

#### CP2 — 화재경보기 누르기
| | |
|---|---|
| 트리거 | CP1 정답 후 자동. 도서관 좌측 벽에 빨간 화재경보기 표시 강조 (반짝) |
| 친구 반응 | 모두 학생을 보며 "어디로 가지?" |
| 정답 행동 | 화재경보기로 이동 → A 버튼 누르기 → 사이렌 울림 |
| 오답/시간초과 시 힌트 | 친구 (랜덤 슬롯): "벽에... 빨간 게 있던 것 같아. 학교에서 본 적..." |
| 정답 후 | 사이렌 울림 + 친구들 학생 쪽으로 모임 |

#### CP3 — 손등으로 문 온도 확인
| | |
|---|---|

| 오답 (손바닥) | 화상 시각 효과 + 친구 (랜덤 슬롯): "아 맞다, 뜨거우면 만지면 안 된다고..." |

|---|---|

| | |
|---|---|

### Map 3: 운동장 (`act1_playground`)

#### CP6 — 안전거리 + 점호
| | |
|---|---|
| 트리거 | 학교 건물 밖 도착 |
| 시각 단서 | 위쪽 멀리 화재 건물 (연기 솟음) / 아래쪽 안전 지점 (선생님이 학생들 모으는 자리) |
| 친구 반응 | 친구A: "여기 가까이서 봐도 돼?" (위험한 호기심) |

| 오답 (건물 가까이) | 위험 표시 (빨간 경고) + 힌트 (유나): "선생님이 멀리 떨어지라고 했었어..." |
| 정답 후 | 친구 5명 + 학생 = 6명이 안전 지점에 모임 → 1막 클리어 |

## 게임 메커닉 메모

- **카메라**: 1280×720 뷰포트가 1280×1280 맵에서 학생을 따라다님 (`startFollow`)
- **카운트다운**: 각 체크포인트 10초. UI 우상단 표시
- **힌트 트리거**: 1차 실패 OR 카운트다운 3초 남았는데 미입력
- **친구 idle 애니메이션**: 가만히 있어도 살짝 떨림 (Y축 sine wave 5px)
- **친구 follow**: 학생이 정답 행동 후 친구들이 학생 위치로 자동 이동 (간단한 lerp)
- **A 버튼**: 가까운 인터랙션 오브젝트 활성화 (mesa 패턴)
